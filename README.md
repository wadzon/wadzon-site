# WADZON site

Простая статическая страница-визитка музыкального проекта WADZON.

## Структура

- `index.html` — главная страница.
- `styles.css` — стили сайта.
- `script.js` — блокировка клика по временным ссылкам-заглушкам, если они появятся позже.
- `assets/` — изображения проекта.
- `assets/ne-delay-potishe.mp3` — локальный MP3 для встроенного плеера на странице.

## Ссылки на площадки

На странице заполнены VK, YouTube, BandLink, Telegram, TikTok, MAX, Rutube, Avito, личный Telegram, личный VK, личный MAX и почта.

Сайт рассчитан на деплой как HTML/CSS/JS из корня репозитория `/`.
